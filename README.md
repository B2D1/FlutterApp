# Future，一款基于 Flutter 的待办事项应用
> 该项目作为本人移动端期末课程设计，供大家学习参考
## 实现功能：
- 基于 `SQLite3` 的基础 `CURD`
- 实现基础布局：如Listview，Card，Padding、Row...
- 监听用户输入，日期选择
